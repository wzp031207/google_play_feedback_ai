# 简历项目描述

## 中文版

**基于数据分析与 LLM 的 Google Play 用户反馈洞察及运营建议助手**

- 基于 Google Play 应用与游戏评论真实数据，搭建用户反馈分析、问题主题发现、AI 摘要生成与检索式问答一体化分析助手，处理评论数据超过 100 万条。
- 自动识别多张原始 CSV 的真实 schema，统一映射评论文本、评分、时间、标题、分类、安装量等字段，完成数据清洗、缺失处理、重复评论去重与分析底表构建。
- 设计评论量趋势、平均评分趋势、低分占比、标题评分下滑排行、情绪异常样本、评分区间关键词差异等分析模块，输出可复用聚合结果与数据质量报告。
- 基于低分评论实现 `TF-IDF + NMF` 主题发现，生成主题名、核心关键词、代表评论、评论量、均分、近 7/30 天趋势与优先级分数，支持业务快速定位高优先级问题。
- 构建带 fallback 的 LLM 洞察生成能力：无 Key 时使用规则化摘要，有 Key 时调用 OpenAI 生成周报摘要、产品建议、运营建议与风险预警。
- 使用 Streamlit 搭建中文交互式可视化看板与检索式问答助手，支持按实体类型、分类、标题、时间过滤反馈数据，适合作品集展示与面试演示。

## English Version

**Google Play User Feedback Insight and Operations Recommendation Assistant Powered by Analytics and LLM**

- Built an end-to-end analytics assistant on real Google Play app and game review data, covering data profiling, feedback analysis, issue discovery, AI summarization, and retrieval-based Q&A over more than 1 million reviews.
- Automatically inspected raw CSV schemas and unified review text, rating, date, title, category, downloads, and metadata into a reusable analytical base table with cleaning, deduplication, missing-value handling, and data quality reporting.
- Developed reusable analysis modules for review volume trends, rating trends, low-rating share, title-level rating drop ranking, sentiment mismatch detection, and keyword differences across rating buckets.
- Implemented low-rating issue discovery using `TF-IDF + NMF`, producing topic labels, core keywords, representative reviews, review volume, average rating, recent 7/30-day trend, and a business-oriented priority score.
- Added LLM-powered weekly summaries and recommendation generation with a robust fallback strategy: rule-based outputs when no API key is available, and OpenAI-generated natural language insights when configured.
- Built a Chinese Streamlit dashboard and retrieval-based QA assistant for portfolio and interview demos, enabling interactive filtering by entity type, category, title, and time range.
